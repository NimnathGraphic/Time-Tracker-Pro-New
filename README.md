# TimeTrack Pro (AU) - Employee Time Management System

TimeTrack Pro is a robust, single-file HTML application designed for small to medium-sized businesses in Australia to manage employee time tracking, particularly across multiple shop locations. Built with a focus on simplicity, role-based access control, and data persistence, this system provides a professional and user-friendly solution for owners and managers.

## Key Features

*   **Role-Based Access Control (RBAC):** Separate, secure logins for **Owner** and **Manager** roles.
    *   **Owner:** Full system access, shop management, and the ability to grant/revoke "Full Access" to managers.
    *   **Manager:** Restricted view to only employees in their assigned shop, simplified employee addition, and time entry management.
*   **Local Data Persistence:** All employee, shop, and time entry data is securely saved in the browser's **Local Storage**, ensuring data remains intact between sessions.
*   **Australian Localization:** Features an Australian-themed UI, AU date/time formats, and a live clock.
*   **Advanced Reporting:** Generate detailed reports with filtering by employee and shop. Reports include:
    *   Total Work Hours, Total Break Hours, and a separate calculation for **Weekend Hours**.
    *   Export functionality for **PDF** and **Excel (CSV)** formats.
*   **Flexible Time Entry:** Supports both **Live Clock** entries (Clock In/Out, Break Start/End) and **Manual Time Entry** with a quick "Today" button for date selection.
*   **Cross-Shop Employee Management:** Logic supports employees who may work across different shops, with managers only seeing relevant staff.
*   **Professional UI/UX:** Clean, modern, and fully **mobile-responsive** design for use on any device.

## Getting Started

1.  **Download:** Download the `timetrack_pro_v6_clean.html` file.
2.  **Open:** Open the file directly in any modern web browser (Chrome, Firefox, Edge, etc.).
3.  **Login:** Use the default credentials to access the system:
    *   **Username:** `owner`
    *   **Password:** `Owner@123`
4.  **Customize:** Immediately change the default password in the **Settings** tab and begin adding your shops and employees.
